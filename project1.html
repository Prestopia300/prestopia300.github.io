Piano Cubes

A 3D Unity Game Engine Project
Works with MIDI files

This project was created as an extention of previous code I had written that works with MIDI files. The goal is to combine code with music in an interesting way. The Purpose of Piano Cubes is to make (a 3D version of Piano Tiles) game where a person can play piano to a song in a 3D space. Tiles fall and hit the piano when note is played, and players can play along with the notes as they fall. This project contains a scene node hierarchy that can be maipulated in the form of segmented piano keys for octaves 1-2, 3-4, and 5-6. Color change and Unity particles when note is played. 2 view ports, one normal view (which can zoom, tumble and pan), and one orthogonal view. It uses custom diffuse illumination shaders and textures with fixed directional light for the walls of the "meme cube".

Above, is an example of the song All Star being played. The falling blocks represent the song, read from a MIDI file, and the piano key is being played, showing a color and Unity particles.

Above, is the pause menu, which holds options to stop, choose a song, and resume. The Stop button stops the song, eraseing all blocks and erasing future falling blocks in queue. Choose song (clicking "Super Smash Bros" or "All Star") will start the song from the beginning, replacing the previous queue with new notes. Resume resumes the song, or Creative mode if no songs are playing :)
